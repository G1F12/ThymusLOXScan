# Target Lab Search Plan

This is a planning document, not a completed lab search.

## Search Criteria

- Thymus aging expertise.
- Thymic epithelial cell or mTEC biology.
- Immunosenescence or stromal aging.
- ECM/stroma or LOX/LOXL2 biology.
- IHC/IF/RNAscope capability.
- Mouse thymus tissue access.
- Europe/Poland/EU preferred when otherwise equally relevant.
- Existing publications using thymus sections, TEC markers, mTEC markers, or aging histology.

## Prioritization

1. Labs with direct mouse thymus aging and TEC/mTEC histology experience.
2. Labs with strong thymic epithelial cell biology and access to young/aged mouse thymus tissue.
3. Histology or RNAscope cores with mouse thymus experience.
4. ECM/LOX biology labs with staining expertise and interest in thymic stroma.

## Outreach Workflow

1. Identify public lab page and relevant publications.
2. Record public contact URL and public email only if intentionally listed.
3. Send short initial email with one-page pitch attached or linked.
4. If interested, send detailed proposal and technical appendix.
5. Follow up after 7-10 days if no response.
