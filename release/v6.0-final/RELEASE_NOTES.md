# v6.0 final computational preprint package

Version 6 is the consolidated final manuscript package. No new scientific analysis was introduced during final assembly. GSE240016 remains the primary discovery dataset. GSE231906 is `DETECTION_CONTEXT_ONLY`; external evidence remains mixed or contextual; and the mTEC1 `Loxl2` finding remains candidate-level. Wet-lab validation is necessary.

The final presentation polish adds three reader-facing figures assembled only from frozen authoritative tables, removes internal phase jargon from the scientific narrative, clarifies the internal computational robustness categories, and provides clean HTML, PDF, and DOCX formats. The public PDF includes numbered footers.

Two isolated `FULL_LOCAL` attempts reproduced the available Python core but remained partial because specified external inputs and the R/Bioconductor environment were unavailable. `RELEASE_DERIVED` checks completed without numerical invariant failures. The release therefore does not claim complete clean-room reproduction of every historical workflow.

Historical versions remain available for provenance but are superseded for current interpretation by this manifest-defined snapshot.
